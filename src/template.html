Employee Tracker

Created the employee tracker application using Angular, RESTFUL API, and MongoDB. With the help of this application, every employee of the company will be able to search for any employee within the organization, retrieve the data from the database and update the outdated information for that person. Also, employees will be able to create new positions and assign that newly created position to any employee. This web app helps to keep the track of employees within any company and makes sure that information is up to date for each employee.
