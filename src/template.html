Smart Brain Web Application

I have created a complex image recognition app using HTML5, CSS3, Bootstrap 5, JavaScript, React + React Hooks, Node.js, Express.js, NPM, RESTful API Design, SQL and PostgreSQL. The application allows users to register, sign in and upload any image and detect face if exists. The application uses AI Clarifai API to detect a face on the uploaded image. In addition, users can update their information on the profile page. Furthermore, there is an administrator who has complete control of the application. The administrator can monitor the activities as well as delete, create and update information about every user in the system.
